# Planner

This is a simple calendar application that allows the user to save events for each hour of the day their work day.  This application runs in the browser and automatically displays the current date.

## Key Features

* Displays the current date in the format of day of the week, month, day of month and year.

* Includes the hours for a typical 9:00am to 5:00pm workday.

* Allows the users to enter plans or tasks in an hour block, hit save, and the content will stay when the page is reloaded.

* The hour will be grey in color for time that has past, red in color for the current hour and green in color for the future time.

### How to Use

* To use this application, open it in the browser.

* The user will click in the block of time of the hour they would like to designate their plans or task.

* After they enter their task, they will click the save button.

* Once the content is saved, it will remain whenever the page reloads, until it is deleted and saved again.

* The current date of the time of login will automatically be displayed at top of the page.
